This repository holds all the GitHub issue [labels](https://github.com/abhijithvijayan/labels/labels) I use across other repositories.

I use [import-github-labels-cli](https://www.npmjs.com/package/import-github-labels) to sync the labels across other repositories with command
> `npx import-github-labels sync`
    
View more about the CLI [here](https://github.com/abhijithvijayan/import-github-labels-cli)

This repository was created using [create-remote-repo-cli](https://www.npmjs.com/package/create-remote-repo) with command
> `npx create-remote-repo labels`
    
View more about the CLI [here](https://github.com/abhijithvijayan/create-remote-repo-cli)
